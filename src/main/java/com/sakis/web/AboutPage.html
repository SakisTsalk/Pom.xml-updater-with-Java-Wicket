How it Works

Pom Updater is a tool intended to make the lives of developers who use Apache Maven as their project managment and comprehension tool easier. Just upload the project managment file(pom.xml) and then you can see which of your dependencies and plugins are outdated and choose which ones you want automatically being updated. Then you just download the updated pom.xml file and let your IDE import the latest versions of your dependencies and plugins found online.
